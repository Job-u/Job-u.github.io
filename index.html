BEDTE: AI and Speech Recognition for Preserving the Endangered Casiguran Agta Language

HTML CSS PHP JavaScript Python Flask MySQL

BEDTE is a capstone project aimed at preserving the endangered Casiguran Agta language through the power of AI and speech recognition technology. The website allows users to interact by speaking or typing in English, Tagalog, or Casiguran Agta, providing translations and pronunciation guides for each input.
